# RealMinecraft

**LingyunCraft** *Project*

**Author:** 贺兰星辰 shaokeyibb

**Co-Contributor:** EntityParrot_

- - -

RealMinecraft 插件能给服务器添加几种新的有趣特性:

+ 使用桶舀水/岩浆时, 有几率使桶破裂.
+ 死亡的时候, 有几率触发惩罚.
+ 在雷雨天戴着金属头盔时, 有几率被闪电击中.
+ 隐藏功能 (见源代码).
